# Security Policy

## Supported Versions
This repository contains personal problem-solving notes and logs for Codeforces. All content is maintained privately for personal study purposes. No official support or public solution guidance is provided.

## Reporting a Vulnerability
If you find any issue related to security (for example, accidental exposure of sensitive data), please contact the repository owner via GitHub [issues](https://github.com/trihajikhr/codeforces-solving/issues) or email at `trihajikhr@gmail.com`.

## Data Privacy
- This repository contains only personal notes and solved problems references.
- Do **not** share login credentials or sensitive personal data in any issues or pull requests.
- If any sensitive data is accidentally exposed, please contact the repository owner immediately.

## Responsible Disclosure
Please report any security issues responsibly. Do not exploit vulnerabilities for personal gain.
